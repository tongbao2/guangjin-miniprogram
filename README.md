# 广金网小程序

金属行情导航微信小程序，快速访问 10 个热门金属行情网站。

## 功能特点

- ✅ **简洁工业风格**：深色主题，工业风格配色
- ✅ **10 个金属行情网站**：
  - 伦敦行情、南海铝锭、佛山报价
  - 废旧市场、报价预测、今日黄金
  - 金属快讯、中金信息、7x24 快讯、历史报价
- ✅ **快速访问**：点击即可打开网页浏览
- ✅ **搜索功能**：支持关键词过滤
- ✅ **现代交互**：卡片设计、动画效果、响应式布局

## 使用方式

### 微信小程序开发工具

1. 安装微信小程序开发工具
2. 导入 `广金网小程序` 文件夹
3. 点击 "编译" 预览小程序

### 发布步骤

1. 注册小程序账号（企业或个人）
2. 上传代码至微信官方平台
3. 提交审核，等待发布

## 适配建议

由于部分网址可能包含小程序不允许访问的域名，你需要：

1. **域名备案**：在小程序后台配置业务域名
2. **https 支持**：所有链接必须使用 https 协议（目前 10 个网站都支持）
3. **隐私合规**：确保用户知情同意

## 其他说明

- 配色采用工业金属风格：深蓝底色 + 金色点缀
- 支持暗黑模式，降低视觉疲劳
- 卡片设计让操作一目了然
